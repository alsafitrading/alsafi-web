# AL SAFI GENERAL TRADING LLC - Website

A professional, modern, and SEO-optimized website for AL SAFI GENERAL TRADING LLC, showcasing weighing scales, POS solutions, automation systems, and packaging solutions.

## Features

- ✅ **Modern & Professional Design** - Clean, responsive layout with smooth animations
- ✅ **SEO Optimized** - Meta tags, semantic HTML, structured data, alt text for images
- ✅ **Performance Optimized** - Lazy loading, debounced/throttled events, optimized assets
- ✅ **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices
- ✅ **Accessible** - Keyboard navigation, ARIA labels, semantic HTML
- ✅ **Interactive** - Smooth scrolling, animated counters, mobile menu

## File Structure

```
alsafi-web/
├── index.html          # Main HTML file
├── styles.css          # All CSS styles
├── script.js           # JavaScript functionality
├── robots.txt          # SEO robots file
├── sitemap.xml         # SEO sitemap
└── README.md           # This file
```

## Getting Started

1. Simply open `index.html` in a web browser
2. For local development, you can use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1e40af;
    --secondary-color: #7c3aed;
    --accent-color: #f59e0b;
    /* ... */
}
```

### Content
- Update text content in `index.html`
- Replace placeholder images with actual product images
- Update contact information in the contact section

### Images
Replace placeholder images with actual product images. Make sure to:
- Use descriptive alt text for SEO
- Optimize images (WebP format recommended)
- Use appropriate sizes (max 1200px width recommended)

## Notes

- Replace placeholder image URLs with actual product images
- Update contact information (email, phone, address)
- Update sitemap.xml with your actual domain
- Add your company logo as favicon.ico
- Consider adding Google Analytics for tracking
